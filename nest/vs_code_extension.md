<br />

2. Material Icon Theme (파일 아이콘 설정)

<br />

3. Prettier (Code formatter)

```
1. 설정 (Command(ctrl) + ,) 에 들어간다.

2. "save"를 검색한다.

3. "Editor: Format On save" (저장 시 자동 포멧) 체크

4. "formatter"를 검색한다.

5. Text Editor -> Formatting 에 들어간다.

6. Editor를 Prettier - Code formatter로 변경한다.
```
